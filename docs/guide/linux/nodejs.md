# NodeJS

> nvm 

[github](https://github.com/nvm-sh/nvm)

```
# 安装lts 
$ nvm install --lts
# 安装指定版本
$ nvm install v14.18.3
# 列出所有远程可用版本
$ nvm ls-remote
# 列出所有本地安装版本
$ nvm ls
# 切换使用指定的版本
$ nvm use v14.18.3
# 改变默认版本
$ nvm alias default v14.18.3
# 删除已安装的指定版本
$ nvm uninstall v14.18.3
```

> nrm 

[github](https://github.com/Pana/nrm)

```
# 全局安装
$ npm install -g nrm
# 列出所有源
$ nrm ls
# 使用指定源
$ nrm use taobao
```
